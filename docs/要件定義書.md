# Brewia 要件定義書

## 用語定義

| **和名** | **英名** | **定義** |
| --- | --- | --- |
| 豆 | Bean | コーヒー豆、およびコーヒー豆の基本情報。 |
| 抽出 | Brew | 豆からコーヒーを抽出するレシピ、抽出したコーヒー、およびそれらの組み合わせ。 |
| レシピ | Recipe | 豆からコーヒーを抽出する方法、およびそのための設定値。 |
| カップ | Cup | 豆から抽出したコーヒー、およびその味わい。 |
| フレーバー | Flavor | コーヒーの味わいにおいて感じられる風味。 |

## 機能要件

### 豆管理機能

この機能では、豆情報の一覧表示・詳細表示・作成・コピー・編集・削除を実行できる。実行対象となる豆の情報は、以下である。

| **和名** | **英名** | **概要** |
| --- | --- | --- |
| 名称 | Name | 豆の名称。 |
| 生産国 | Country | 豆が生産された国。 |
| 生産地域 | Region | 豆が生産された地域。 |
| 生産農園 | Farm | 豆が生産された農園。 |
| 生産処理 | Process | 豆の生産処理方法。 |
| 品種 | Variety | 豆の品種。 |
| 焙煎度 | Roast | 豆の焙煎度。 |
| 焙煎所 | Roaster | 豆の焙煎所。 |
| メモ | Note | 豆に関する自由記述。 |
| 作成日時 | Created | 豆情報を登録した日時。 |
| 編集日時 | Updated | 豆情報を編集した日時。 |
- 生産国は基本的に、国旗と共に表示する。ブレンドの場合は `🏳️‍🌈` を表示する。
- 焙煎度は、以下の8つのレベルで構成される。これは、『[What Color is Your Coffee? | 25, Issue 21](https://sca.coffee/sca-news/25/issue-21/what-color-is-your-coffee)』『[A universal color curve for roasted arabica coffee](https://www.nature.com/articles/s41598-025-06601-w)』を元に定義したものである。
    - ライトロースト（Light Roast）： `#BA8D5D`
    - シナモンロースト（Cinnamon Roast）： `#AD7C4E`
    - ミディアムロースト（Medium Roast）： `#9E6C41`
    - ハイロースト（High Roast）： `#8D5D37`
    - シティロースト（City Roast）： `#7B4F2F`
    - フルシティロースト（Full City Roast）： `#68432B`
    - フレンチロースト（French Roast）： `#533728`
    - イタリアンロースト（Italian Roast）： `#3D2D27`
- 生産処理は、以下の方法から選択する。
    - ナチュラル
    - ウォッシュト
    - パルプトナチュラル
    - レッドハニー
    - イエローハニー
    - ホワイトハニー
    - アナエロビック
    - その他

### 抽出管理機能

この機能ではコーヒーの抽出レシピ、および抽出したカップに関する情報の一覧表示・詳細表示・作成・コピー・編集・削除を実行できる。実行対象となる抽出の情報は、以下である。

| **和名** | **英名** | **概要** |
| --- | --- | --- |
| 豆量 | Bean Weight | 豆の重量（g）。 |
| 挽き目 | Bean Grind | 豆の挽き目（クリック数）。 |
| 湯量 | Bean Weight | 湯の重量（g）。 |
| 湯温 | Bean Temp | 湯の温度（℃）。 |
| 抽出ステップ | Brew Steps | 注湯のステップ。 |
| 香り | Aroma | カップの香り。 |
| 酸味 | Acidity | カップの酸味。 |
| 甘味 | Sweetness | カップの甘味。 |
| 質感 | Body | カップの質感。 |
| 総合点 | Overall | カップの総合点。 |
| メモ | Note | 抽出に関する自由記述。 |
| 作成日時 | Created | 抽出情報を登録した日時。 |
| 編集日時 | Updated | 抽出情報を編集した日時。 |
- 豆量と湯量から抽出比率（Brew Ratio）を表示する。
- 抽出ステップでは、注湯時間（s）ごとの湯量（g）を保存する。
- 抽出ステップは、横軸を時間、縦軸を湯量とする折れ線グラフで描画する。
    - グラフの各ステップには、ステップ数を記載したポイントを表示する。
    - 各ステップのポイントを示すメモリ線を、ステップの時間と湯量から動的に表示する。
- カップの味わいは、レーダーチャートで描画する。

### フレーバー管理機能

この機能では、コーヒーの風味情報の一覧表示を実行できる。実行対象となるフレーバーの情報は、以下である。

| **和名** | **英名** | **概要** |
| --- | --- | --- |
| 名称 | Name | フレーバーの名称。 |
| カテゴリ | Category | フレーバーのカテゴリ。 |
| サブカテゴリ | Subcategory | フレーバーのサブカテゴリ。 |
| 作成日時 | Created | フレーバーを登録した日時。 |
| 編集日時 | Updated | フレーバーを編集した日時。 |
- フレーバーは、抽出情報を作成する際に表示・選択可能とする。

### ダッシュボード機能

- 総抽出数（Total Brews）と総豆数（Total Beans）を表示する。
